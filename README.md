# Teachers-timetable-management-system

# Installation:
1. Clone repository.
3. Ensure all dependencies are installed.
2. Run `python main.py` on project directory.

# Usage:
1. Add instructors
2. Add rooms
3. Add subjects
4. Add sections
5. Generate solution
6. Export or View Result

# Dependencies:
1. Numpy
2. PyQT5
3. psutil
